============ HOWTO ===========
Steps:
> create the project
  $ mix phx.new your_app --no-install --app your_app --database sqlite3 \
     --no-live --no-assets --no-html --no-dashboard --no-mailer --binary-id
> Create schema
  $ mix phx.gen.context Data Gig gigs title:string description:string songs:string
> Create database 
  $ mix ecto.migrate
